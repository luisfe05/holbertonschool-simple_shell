# # Simple Shell (hsh) – Holberton School Project

This project is part of the Holberton School curriculum.

In this project, we built our own version of a UNIX command line interpreter, called `hsh`.  
It reads user input, executes commands, and behaves similarly to `/bin/sh`.

Right now, our `hsh` handles:
- Executing commands with absolute paths (e.g. `/bin/ls`)
- Executing commands using the `PATH` variable
- Interactive mode (user input)
- Non-interactive mode (piped input)
- Process creation with `fork`
- Program execution with `execve`
- Waiting for child processes with `wait`
- Basic error handling (command not found)
